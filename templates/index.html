Purpose and Our Work Ethics

The purpose of SpeechSign Software is to ease online communication between Specially-Abled group of citizens and to create a convenient and easy-to-use application for everyone. The system is based on the Deep Learning Model to detect the Sign Language, and also to convert speech to sign language

About us

Documentation

We have ensured that the documentation is precise and detailed for any user to start using our software.

Analysing Packages

We have tried and used multiple packages manages and dependancy resolver like depandabot, pip, conda and more, for a stable project.

Designing

While designing of our project we have ensured that the UI/UX for the software remains user friendly and easy to learn.

Security

To ensure secure authentication and securing user data we have relied on Google Firebase and Google Firestore, which provides a secure, scalable authentication and data storage platform.

Features

SpeechSign focuses on a feature rich environment for their user that is user friendly. We have made sure that our user gets all the necessary feature to converse with a specially-abled person and get a realtime analysis of the statistics.

Speech to Sign Language Support

To convert Speech to Sign, we first converted speech into text using WebkitSocketRecognition JS package, and using the text converted those to Sign Languega.

  • Converting Speech with on-click button to text.
  • Converting text to sign language using fetched images.
  • Stacking the fetched images and making a video out of it.
